How to add the Students and Professors in your institution. Courses will be automatically generated from the users' courseLists.

  1. You need to import a file in JSON format as follows:
    2.             {
                "0": //user #
                    {
                        "email": "email@example.com",
                        "isProf": "false",
                        "firstName": "Alex",
                        "lastName": "James",
                        "password": "some previously generated password"
                        "courseList": 
                            {
                                "0": "/courses/COMP232-A", //Where COMP232-A is the name of the course. 
                                "1": "/courses/SOEN287-Q",
                                "2": "/courses/COMP249-D",
                                "3": "/courses/COMP233-E"
                            }
                    },
                "1":
                    {
                        "email": "other@example.com",
                        "isProf": "true",
                        "firstName": "John",
                        "lastName": "Wick",
                        "password": "123456"
                        "courseList": 
                            {
                                "0": "/courses/COMP232-A", //Different sections should be listed as different courses.
                                "1": "/courses/COMP232-B",
                                "2": "/courses/COMP249-D",
                            }
                    },
                etc...
            }
  2. Note that all fields are required.
